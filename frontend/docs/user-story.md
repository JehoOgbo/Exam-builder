### 1. Account & Security
* *As an educator*, I want to create an account so that my exams are securely saved and tied to my identity.
* *As a returning educator*, I want to securely log in with my credentials so that I can access my private dashboard and resume my work.

### 2. Dashboard Management
* *As an educator*, I want to view a centralized dashboard of all my created exams so that I can easily find and manage my historical assessments.
* *As an educator*, I want the ability to permanently delete an outdated or incorrectly authored exam so that my dashboard stays clean and organized.

### 3. Exam Creation (The Builder)
* *As an educator*, I want to input high-level metadata (such as School Name, Subject, Course Code, and Time Allocated) so that the exam paper has a professional, informative header.
* *As an educator, I want to freely add, label, and remove distinct **Sections* (e.g., "Section A: Multiple Choice", "Section B: Theory") so that the exam is structurally organized.
* *As an educator, I want to dynamically add **Questions* under any section so that I can flesh out the assessment content.
* *As an educator, I want to attach multiple **Options* to my questions and visually designate the correct answer so that I can build multiple-choice formats effectively.
* *As an educator*, I want to save the entire complex exam structure to the cloud with a single click so that I don't lose my intricate work.

### 4. Review & Export (Preview/Print)
* *As an educator*, I want to "Preview" an exam I previously created so that I can review its final layout without accidentally editing any fields.
* *As an educator, I want to export my exam to a clean, paper-ready **PDF format* (which automatically strips away buttons, empty bubbles, and bulky inputs) so that it is instantly ready to print and distribute to my students in a classroom
